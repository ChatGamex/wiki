# Outfits & Colors

Customize your character's appearance with outfits and color slots.

## Outfits

Use `!outfit <name>` to change your skin:

| Outfit ID | Display Name |
|-----------|-------------|
| `civilian_male` | Civilian Male |
| `civilian_female` | Civilian Female |
| `archer_male` | Archer Male |
| `archer_female` | Archer Female |
| `mage_male` | Mage Male |
| `mage_female` | Mage Female |
| `knight_male` | Knight Male |
| `knight_female` | Knight Female |
| `monk_male` | Monk Male |
| `noble_male` | Noble Male |
| `noble_female` | Noble Female |

Default: `civilian_male`.

## Colors

Use `!color <slot> <R,G,B>` to change clothing colors. Values 0-255.

| Slot | Description | Default |
|------|-------------|---------|
| `head` | Head/hair color | `66, 62, 56` |
| `primary` | Primary clothing color | `111, 0, 16` |
| `secondary` | Secondary clothing color | `34, 29, 25` |
| `detail` | Detail/accent color | `112, 0, 18` |

### Examples

```
!color head 255,200,100
!color primary 0,150,200
!color secondary 50,50,50
!color detail 255,0,0
```
